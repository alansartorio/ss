## Input data

Input data has the following format:

```
W (table x size)
H (table y size)
HD (hole diameter)
D (ball diameter)
M (mass)
N (number of balls)
id1 x1 y1 vx1 vy1
id2 x2 y2 vx2 vy2
idN xN yN vxN vyN
```


## Output data

Output data has the following format:

```
t0
N0 (number of balls on table at t0)
id1 x1 y1 vx1 vy1
id2 x2 y2 vx2 vy2
idN xN0 yN0 vxN0 vyN0
t1
N1 (number of balls on table at t1)
id1 x1 y1 vx1 vy1
id2 x2 y2 vx2 vy2
idN xN1 yN1 vxN1 vyN1
```
